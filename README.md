🟦 Dynamic Color Changer Boxes

📌 Project Description :

This is a simple and interactive web project where the colors of boxes change dynamically in two ways:

🎨 On Page Refresh – Every time the page reloads, new random colors are generated for all boxes.

🖱️ On Click – Clicking on any box instantly changes its color to a newly generated random color.

The project is built using HTML, CSS, and JavaScript and demonstrates fundamental front-end development concepts such as:

➡️ DOM Manipulation

➡️ JavaScript Event Handling

➡️ Random Color Generation

➡️ Interactive UI Behavior
